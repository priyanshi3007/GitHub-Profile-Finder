🎨 **Design Tools**: Figma, Photoshop

💻 **Tech Stack**: HTML, CSS, JS, jQuery, Bootstrap, API, React, GitHub API

🚀 **Key Features**:

- Monitor your GitHub activity
- Clean and user-friendly design
